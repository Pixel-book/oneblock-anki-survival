# OneBlock Anki Survival

Anki 单方块生存是一个 Minecraft Bedrock 行为包。主世界会变成虚空，玩家从一个核心方块开始生存。

每刷新一次核心方块会消耗 1 点个人刷新余额。余额为 0 时，核心不会继续刷新；完成一套 5 题 Anki 题组后，个人余额增加 5。

下界和末地保持原版生成。玩家仍然需要自己建下界门、进入下界、获得烈焰棒、合成末影之眼、前往远处祭坛、手动填眼、进入末地并击败末影龙。

## 导入

建议新建世界使用。

构建后导入：

```text
releases/OneBlockAnki_Addon.mcaddon
```

`.mcaddon` 会同时导入行为包和资源包。若你已经导入过 0.1.0 测试版，建议先在 Minecraft 存储里删除旧的 OneBlock Anki Survival 行为包/资源包，再导入新版，避免旧 UUID 缓存残留。

已有世界可能保留旧区块，不保证完全虚空化。Xbox 成就是否触发不承诺，本项目只保证通关链条设计可达。

## 玩法

1. 进入世界后站在核心方块上。
2. 破坏核心方块，系统按阶段刷新新方块。
3. 余额用完后完成 Anki 题组获得余额。
4. 阶段随世界总刷新次数推进。
5. 水、岩浆、铁、黑曜石等关键资源有无感保底。
6. 末地祭坛岛在约 1500 格外，前期靠近会出现空间波动提示。
